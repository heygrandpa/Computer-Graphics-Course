# B Spline Curve and Surface

## Required

1. OpenGL 3.3+ (GLSL 3.3+)
2. GLFW
3. GLM

## Features

1. Light Shading
2. Movement (Using Keyboard `W` `A` `S` `D` `UP` `DOWN` `LEFT` `RIGHT`)
3. B Spline Surface with Animation
